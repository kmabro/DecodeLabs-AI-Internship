# Rule-Based AI Chatbot

A simple rule-based chatbot built with Python as part of the DecodeLabs internship program.

## What it does

- Responds to greetings like hello, hi, hey
- Answers basic questions (name, how are you)
- Tells jokes
- Says goodbye when you type bye or exit
- Runs in a loop until you quit

## How to run

```
python chatbot.py
```

## What I used

- Python 3
- if-else conditions
- random module for varied responses

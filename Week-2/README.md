# Data Classification Using AI

A k-Nearest Neighbors classification model built with Python as part of the DecodeLabs internship program.

## What it does

- Loads the Iris flower dataset (150 samples, 3 classes, 4 features)
- Splits data into 80% training / 20% testing
- Trains a k-NN classifier (k=5) on the training data
- Evaluates the model with accuracy, classification report, and confusion matrix
- Shows sample predictions for the first 5 test samples

## How to run

```
python classification.py
```

## What I used

- Python 3
- scikit-learn (load_iris, train_test_split, KNeighborsClassifier)
- accuracy_score, confusion_matrix
- classification_report and confusion_matrix from scikit-learn

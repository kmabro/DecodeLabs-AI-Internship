# AI Recommendation Logic

A simple recommendation system that matches user interests to items using similarity scoring, built with Python as part of the DecodeLabs internship program.

## What it does

- Presents categories (Movie, Book, Song) and lets the user choose one
- Shows available tags for the selected category in aligned columns
- Validates user interests against available tags and requires at least 3
- Scores items using similarity logic (matching interests / total interests)
- Displays the top 3 recommendations with match percentages and matching interests

## How to run

```
python AI_Recommendation_Logic.py
```

## What I used

- Python 3
- No external libraries (standard library only)
